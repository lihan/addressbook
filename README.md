# Address Book #

Sample Google App Engine application in Python 2.7.

## Requirements ##

* Google App Engine SDK
* Python 2.7

## Installation ##

`$ make install`

## Usage ##

### Running the application locally ###

`$ make run`

### Deploying the application ###

`$ make deploy`

### Browsing the application ###

`$ make browse`

## Testing ##

`$ make test`

## Sample data ##

See `sample.csv` file under `data` directory
